AI New Tab

Choose Claude, Gemini, ChatGPT, Grok, or a custom URL — instead of MSN.

Open mode

Save applies changes immediately. Clear only resets this form until you Save. Done closes the tab — if you have unsaved changes, you’ll be asked to save or discard.
